# Proyecto-Generador-Contrasenas
# Generador de Contraseñas Seguras

## Descripción

Este proyecto fue desarrollado como trabajo final de la asignatura **Lógica de Programación** de la carrera de **Ingeniería en Ciberseguridad**.

La aplicación permite generar contraseñas seguras de manera personalizada, permitiendo al usuario seleccionar la longitud de la contraseña y los tipos de caracteres que desea incluir, como letras mayúsculas, letras minúsculas, números y símbolos.

---

## Objetivo

Desarrollar un generador de contraseñas utilizando Python, aplicando los principios de la lógica de programación mediante el uso de variables, estructuras condicionales, ciclos repetitivos y diagramas de flujo elaborados en Raptor.

---

## Tecnologías utilizadas

* Python
* Raptor
* Visual Studio Code
* GitHub

---

## Funcionalidades

* Generación de contraseñas personalizadas.
* Selección de longitud.
* Inclusión de mayúsculas.
* Inclusión de minúsculas.
* Inclusión de números.
* Inclusión de símbolos.
* Validación de datos ingresados por el usuario.
* Garantía de al menos un carácter de cada tipo seleccionado.
